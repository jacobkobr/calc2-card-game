# FunctionFlip

A card-matching game built to strengthen your Calculus II skills. Match integral and derivative questions with their answers, track your scores, and compete across varying difficulty levels.

---

## Technologies Used

- **React**
- **KaTeX**
- **Tailwind CSS**
- **JavaScript**
- **Vite**
- **PostCSS**
